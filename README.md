# torapp
A torrent client<br>

<b>Build:</b>
<pre>
git clone https://github.com/alyanser/torapp
cd torapp
cmake -B build -S .
cmake --build build
</pre>

<b>Planned updates:</b>
<pre>
Support for magnet links, TCP trackers & extension protocol
Ability to choose which files to target first / file priorities
Making the visuals less ugly
</pre>
